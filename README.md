# 📝 Spring Boot 회원 관리 시스템

Spring Boot와 Thymeleaf를 사용하여 만든  
회원가입, 로그인, 회원목록 조회 기능을 포함한 간단한 회원 관리 웹 프로젝트입니다.

---

## ✨ 주요 기능

### ✔ 회원가입
- 아이디, 비밀번호, 이름, 전화번호 입력
- 입력값 유효성 검사 (JavaScript)

### ✔ 로그인
- 세션 기반 로그인
- “ID 기억하기” 기능 (localStorage)

### ✔ 로그아웃
- 세션 invalidate로 로그아웃 처리

### ✔ 회원목록 조회
- 등록된 사용자 전체 목록 조회
- Bootstrap 테이블 UI 적용

### ✔ 공통 배너
- 로그인 여부에 따라 메뉴 자동 변경  
  (홈 / 회원가입 / 로그인 / 회원목록 → 홈 / 로그아웃 / 회원목록)
